Reveal provides feature for using toolbar to add custom elements. Default elements for toolbar are readily available for the users to enable or disable Reveal features.
Custom toolbar helps the user to add button to the toolbar container or to the default toolbar container.

## Default Toolbar elements
Default toolbar elements are listed below which can be used to activate or deactivate features in Reveal.
- Take Screenshot - `addAxisToolToggle()`.
- Enable/Disable Axis Tool - `addTakeScreenshotTool()`.
- Enable/Disable Camera Target on Click - `addCameraTargetOnClickToggle()`.
- Toggle Camera Zoom-past/Zoom-to Cursor - `addZoomPastToCursorToggle()`.
- Fit Camera to Model - `addFitCameraToModel()`.

To create and use default elements in toolbar.

```js runnable
// import { ToolbarTool } from '@cognite/reveal/tools';

const toolbar = new ToolbarTool(viewer, model);
toolbar.addAxisToolToggle();
toolbar.addTakeScreenshotTool();
toolbar.addCameraTargetOnClickToggle();
toolbar.addZoomPastToCursorToggle();
toolbar.addFitCameraToModel();
```

## Add elements
Each toolbar element in Reveal provides flexibilty for the user to add tooltip, icon image and a callback function to trigger when the icon button is clicked.
We add a new element to the toolbar with `addToolbarItem(toolTip: string, backgroundImageUri: string, isToggle: boolean, onClick: () => void)`
- `toolTip` - Tooltip for the Icon button
- `backgroundImageUri` - Background image to be placed onto icon button. The image needs to be in string with `base64` format.
- `isToggle` - Is the icon button used as toggle
- `onClick` - Click event callback function which can be used to perform custom functionlity of the user

### Creating new elements for the toolbar
By creating ToolbarTool object, user can add elements to the new toolbar.

```jsx
// import { ToolbarTool } from '@cognite/reveal/tools';

// import iconSet42 from './img/Cognite_Icon_Set-42.png';
// import iconSet54 from './img/Cognite_Icon_Set-54.png';
// import iconSet63 from './img/Cognite_Icon_Set-63.png';

const toolbar = new ToolbarTool(viewer);

const callbackMsg = () => (
    alert("result called")
);

toolbar.addToolbarItem('Explode View', iconSet42, true, callbackMsg);
toolbar.addToolbarItem('Maps', iconSet54, false, callbackMsg);
toolbar.addToolbarItem('Settings', iconSet63, true, callbackMsg);
```

## Toolbar Position
We can set the position of the toolbar to either of the below position within the Reveal window using `setPosition(position: ToolbarPosition)`.
- `top`
- `bottom`
- `left`
- `right`

```jsx
toolbar.setPosition('top');
```
:::note
Default position of toolbar is at the Bottom of the window.
:::
