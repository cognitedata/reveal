Reveal provides a toolbar to add custom elements. Default elements for toolbar are readily available for the users to enable or disable Reveal features. The toolbar can be customized to help the user to add custom buttons.

## Default toolbar elements
Supported default toolbar elements are listed below which can be used to activate or deactivate features in Reveal, or perform common actions.
- Take screenshot - `addTakeScreenshotTool()`.
- Enable/Disable axis tool - `addAxisToolToggle()`.
- Enable/Disable camera target on click - `addCameraTargetOnClickToggle()`.
- Toggle camera zoom-past/zoom-to Cursor mode - `addZoomPastToCursorToggle()`.
- Fit camera to model - `addFitCameraToModel()`.

These can be added to the toolbar on construction:

```js runnable
// import { ToolbarTool } from '@cognite/reveal/tools';

const toolbar = new ToolbarTool(viewer, model);
toolbar.addAxisToolToggle();
toolbar.addTakeScreenshotTool();
toolbar.addCameraTargetOnClickToggle();
toolbar.addZoomPastToCursorToggle();
toolbar.addFitCameraToModel();
```

## Add elements
Each toolbar element in Reveal provides flexibilty for the user to add icon image, a callback function to trigger when the icon button is clicked and optional tooltip.
New elements can be added to the toolbar by methods using
`addToolbarButton(backgroundImageUri: string, onClick: () => void, toolTip: string = '')`
- `backgroundImageUri` - Background image to be placed onto icon button. Pass the data URL of the image.
- `onClick` - Click event callback function which will be used to perform custom functionlity provided by the user.
- `toolTip` - Optional tooltip for the Icon button

`addToolbarToogleButton(backgroundImageUri: string, onToggled: () => void, toolTip: string = '')`
- `backgroundImageUri` - Background image to be placed onto icon button. Pass the data URL of the image.
- `onToggled` - Click event callback function which will be used to perform custom functionlity provided by the user.
- `toolTip` - Optional tooltip for the Icon button

### Creating new elements for the toolbar
By creating ToolbarTool object, user can add elements to the new toolbar.

```jsx
// import { ToolbarTool, GeomapTool, MapProviders, MapboxMode, MapboxStyle, MapboxImageFormat } from '@cognite/reveal/tools';

// import geoMapIcon from './img/Map.svg';

const toolbar = new ToolbarTool(viewer);

let map: GeomapTool | undefined;
const mapConfig: MapConfig = {
  provider: MapProviders.MapboxMap,
  APIKey: APIKeys.MapboxAPI,
  mode: MapboxMode.Style,
  id: MapboxStyle.Satellite_Streets,
  tileFormat: MapboxImageFormat.JPG70,
  latlong: {
    latitude: 59.9016426931744,
    longitude: 10.607235872426175
  }
};
const geomapToggle = () : void => {
  if(map === undefined) {
    map = new GeomapTool(viewer, mapConfig);
  }
  else {
    map.dispose();
    map = undefined;
  }
}

toolbar.addToolbarToogleButton(geoMapIcon, callbackMsg, 'Maps');
```

## Toolbar position
We can set the position of the toolbar to either of the below position within the Reveal window using `setPosition(position: ToolbarPosition)`.
- `top`
- `bottom`
- `left`
- `right`

```jsx
toolbar.setPosition('top');
```
:::note
Default position of toolbar is at the bottom of the window.
:::
