To effectivly navigate a 3D model it is often useful to have an helper to show the orientation of the model. The `GeomapTool` is
a configurable overlay that shows the current orientation of the model. It also reacts to clicks, orienting the camera to face
clicked.

The tool provides list of options to set and choose:
- Several map providers namely, Bing maps, Mapbox and Here maps
- Different map mode like Planar, Spherical & Height 
- Models can be placed on specific geolocation using latitude and longitude values

## Select different Map providers

In Geomap tool, you can select different map providers.
- Bing Maps
- Mapbox
- Here Maps

## Mapbox Maps
Configuration of the Mapbox are listed below which can be used to set different option for the maps.
- 'mode': Map tile access mode.
  - MapBoxMode.STYLE & MapBoxMode.MAP_ID.
- 'APIKey': API access token of Mapbox.
- 'id': Map style to be used.
  - MapBoxStyle.STREETS, MapBoxStyle.OUTDOOR, MapBoxStyle.LIGHT, MapBoxStyle.DARK, MapBoxStyle.SATELLITE,
MapBoxStyle.SATELLITE_STREETS, MapBoxStyle.NAVIGATION_DAY, MapBoxStyle.NAVIGATION_NIGHT, MapBoxStyle.NAVIGATION_GUIDE_DAY, MapBoxStyle.NAVIGATION_GUIDE_NIGHT.
- 'format': Map image tile format.
  - MapBoxImageFormat.PNG, MapBoxImageFormat.PNG32, MapBoxImageFormat.PNG64, MapBoxImageFormat.PNG128,
  MapBoxImageFormat.PNG256, MapBoxImageFormat.JPG70, MapBoxImageFormat.JPG80, MapBoxImageFormat.JPG90, MapBoxImageFormat.PNGRAW
- 'latlong': Latitude & Longitude location where the model will be placed on the map.

```jsx runnable
// import { GeomapTool, MapProviders, MapBoxMode, MapBoxStyle, MapBoxImageFormat } from '@cognite/reveal/tools';

const mapConfig = {
  provider: MapProviders.MAPBOXMAP,
  mode: MapBoxMode.STYLE,
  APIKey: "pk.eyJ1IjoidGVudG9uZSIsImEiOiJjazBwNHU4eDQwZzE4M2VzOGhibWY5NXo5In0.8xpF1DEcT6Y4000vNhjj1g",
  id: MapBoxStyle.SATELLITE_STREETS,
  format: MapBoxImageFormat.JPG70,
  latlong: {
      latitude: 59.9016426931744,
      longitude: 10.607235872426175
  }
}
new GeomapTool(viewer, mapConfig);;
```
## Bing Maps
Configuration of the Bing Maps are listed below
- 'APIKey': API access token of Bing Map.
- 'id': The type of the map used.
  - BingMapType.AERIAL, BingMapType.ROAD, BingMapType.AERIAL_LABELS, BingMapType.OBLIQUE, BingMapType.OBLIQUE_LABELS.
- 'latlong': Latitude & Longitude location where the model will be placed on the map.

```jsx runnable
// import { GeomapTool, MapProviders, BingMapType } from '@cognite/reveal/tools';

const mapConfig = {
  provider: MapProviders.BINGMAP,
  APIKey: "AuViYD_FXGfc3dxc0pNa8ZEJxyZyPq1lwOLPCOydV3f0tlEVH-HKMgxZ9ilcRj-T",
  id: BingMapType.AERIAL,
  latlong: {
      latitude: 59.9016426931744,
      longitude: 10.607235872426175
  }
}

new GeomapTool(viewer, mapConfig);
```

## Here Maps
Configuration of the Here Maps are listed below
- 'APIKey': API access token of Here Map.
- 'APICode': Application code token of Here Map.
- 'id': The type of maps to be used.
  - HereMapType.AERIAL, HereMapType.BASE, HereMapType.PANO, HereMapType.TRAFFIC
- 'scheme': Specifies the view scheme. Be aware that invalid combinations of schemes and tiles are rejected. For all satellite,
 hybrid and terrain schemes, you need to use the Aerial Tiles base URL instead of the normal one.
  - HereMapScheme.DAY, HereMapScheme.NIGHT, HereMapScheme.TERRAIN, HereMapScheme.SATELLITE.
- 'format': Map image tile format.
  - HereMapImageFormat.PNG, HereMapImageFormat.PNG8, HereMapImageFormat.JPG.
- 'latlong': Latitude & Longitude location where the model will be placed on the map.

```jsx runnable
// import { GeomapTool, MapProviders, HereMapType, HereMapScheme, HereMapImageFormat } from '@cognite/reveal/tools';

const mapConfig = {
  provider: MapProviders.HEREMAP,
  APIKey: "HqSchC7XT2PA9qCfxzFq",
  APICode: "5rob9QcZ70J-m18Er8-rIA",
  id: HereMapType.AERIAL,
  scheme: HereMapScheme.TERRAIN,
  format: HereMapImageFormat.PNG,
  latlong: {
      latitude: 59.9016426931744,
      longitude: 10.607235872426175
  }
}
new GeomapTool(viewer, mapConfig);
```
